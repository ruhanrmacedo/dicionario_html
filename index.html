Dicionário de Tags HTML

<!DOCTYPE html>

Declara o tipo do documento. É necessário para informar ao navegador o tipo de documento e a versão do HTML.

HTML <time> Tag

Define um horário específico ou intervalo de tempo. O atributo datetime é usado para especificar o formato legível pela máquina.

HTML <img> Tag

Insere uma imagem no documento. Atributos como src e alt são usados para especificar o caminho da imagem e a descrição alternativa, respectivamente. O atributo width é usado para especificar a largura da imagem.

HTML <ul> e <li> Tag

Define uma lista não ordenada. Cada item da lista é definido com a tag li.

HTML <ol> e <li> Tag

Define uma lista ordenada. Cada item da lista é definido com a tag li e a ordem é numerada automaticamente.

HTML <footer> Tag

Define um rodapé para um documento ou uma seção. Pode conter informações como autor, direitos autorais, e links de rodapé.

HTML <video> Tag

A tag videovideo é usada para incorporar um vídeo em um documento HTML.
